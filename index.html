A Campus ID on a desormais la possibilitée de visualiser, d'uploader et de parcourir les projets web de nos élèves.

Avant d'uploader

  • Insérez un fichier install.txt dans votre projet comportant les instructions shell et php d'installation afin que
    nous puissions installer correctement votre projet sur nos serveurs.
    Les langages de scripts gérés :
    • Le PHP
    • Le shell Linux
    • Le shell Windows
  • Insérez un fichier uninstall.txt dans votre projet comportant les instructions shell et php de desinstallation afin que
    nous puissions desinstaller correctement votre projet de nos serveurs.
    Les langages de scripts gérés :
    • Le PHP
    • Le shell Linux
    • Le shell Windows
